# Bitrix24 Webhook Enricher

Асинхронный сервис на базе **FastAPI**, предназначенный для автоматического обогащения карточек сделок в CRM Битрикс24 актуальными данными о курсе валют (USD) в фоновом режиме.

## Описание логики работы приложения
1. При создании сделки в Битрикс24 срабатывает вебхук `ONCRMDEALADD`.
2. Сервис мгновенно принимает запрос, валидирует входящие параметры и запускает фоновую задачу (`BackgroundTasks`), возвращая Битрикс24 статус `200 OK` (что предотвращает повторные отправки хуков из-за таймаутов).
3. В фоновом режиме сервис опрашивает API валют, получает текущий курс рубля к доллару и через REST API Битрикс24 (`crm.deal.update`) записывает это значение в пользовательское поле сделки.

## Установка и настройка зависимостей

### 1. Клонирование репозитория и переход в папку
```bash
git clone <ссылка_на_ваш_репозиторий>
cd b24_webhook_enricher
```
### 2. Создание и активация виртуального окружения
Windows:
```bash
python -m venv .venv
.venv\Scripts\activate.bat
```
Linux/macOS:
```bash
python3 -m venv .venv
source .venv/bin/activate
```
### 3. Установка библиотек
```bash
pip install -r requirements.txt
```
### 4. Нвстройка конфигурации (.env)
Создайте в корне проекта файл .env и укажите в нём параметры подключения:
```ini
# URL входящего вебхука Битрикс24 с правами на crm
BITRIX24_WEBHOOK_URL=https://ваша_компания.bitrix24.ru/rest/1/ваш_секретный_код/

# Идентификатор пользовательского поля в CRM для записи курса (например, UF_CRM_...)
BITRIX_USER_FIELD=UF_CRM_УИД

# URL стороннего API для получения актуальных курсов валют
CURRENCY_API_URL=https://open.er-api.com/v6/latest/USD
```

## Запуск приложения
Для запуска локального сервера выполните команду в терминале:
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```
Приложение станет доступно локально по адресу: http://127.0.0.1:8000

Для локального тестирования входящих вебхуков от Битрикс24 пробросьте локальный порт наружу в публичный интернет с помощью утилиты вроде `localtunnel` или `ngrok`:
```bash
npx localtunnel --port 8000 --host https://localtunnel.me
```

## Настройка вебхука в Битрикс24
Чтобы связать CRM с вашим приложением:
1. Перейдите в Битрикс24 в раздел **Разработчикам** $\rightarrow$ **Другое** $\rightarrow$ **Вебхук**.
2. Выберите тип: Исходящий вебхук.
3. В поле URL обработчика укажите ваш адрес, полученный из туннеля, добавив к нему эндпоинт обработки: `https://<ваш_домен_из_туннеля>/webhook/deal-add`
4. В типе события выберите: Создание сделки (ONCRMDEALADD).

## Пример тела запроса
Битрикс24 отправляет исходящие вебхуки в формате `application/x-www-form-urlencoded` (данные формы). Ниже представлены варианты для эмуляции поведения CRM.

### 1. Тело запроса в формате Form-data (URL-encoded)
Именно в таком виде данные прилетают от Битрикс24. Этот текст можно вставить в Postman (вкладка `x-www-form-urlencoded`) или использовать для быстрых проверок:
```text
event=ONCRMDEALADD&data[FIELDS][ID]=14**Эквивалент в формате JSON (для наглядности структуры):**
```
### 2. Тело запроса в JSON
```json
{
  "event": "ONCRMDEALADD",
  "data": {
    "FIELDS": {
      "ID": 14
    }
  }
}
```
